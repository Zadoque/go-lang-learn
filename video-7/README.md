## Key Concepts

- Else if chains
- Parentheses omission

```go
if num := 42; num > 50 {
    fmt.Println("Above 50")
} else if num < 30 {
    fmt.Println("Below 30")
} else {
    fmt.Println("Between 30-50")
}
```
